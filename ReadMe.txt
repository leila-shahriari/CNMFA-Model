***************************************************************

This repository includes source code and data for the manuscript 
"Model-based clustering of high-dimensional incomplete data via 
contaminated-normal mixtures."

### Folder description

1) Data: includes real data 
2) Functions: includes functions to implement our methodology 
3) Real-data: includes programs to fit CNMFA model to the real 
data and to regenerate figures
4) Simulation: includes programs to conduct simulation studies
